

# Change PPTable data row border opacity

## File: `src/features/post-production/components/PPTable.tsx`

Replace `border-border/50` with `border-border` on the data `<TableRow>` className.

**Before:** `"border-b border-border/50 hover:bg-muted/40 cursor-pointer transition-colors"`
**After:** `"border-b border-border hover:bg-muted/40 cursor-pointer transition-colors"`

Single class change, no other files.
